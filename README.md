## ⚙️ How to Run

1. Clone the repository using:  
    ```bash
    git clone [repository-url]
    ```
2. Navigate to the project directory:  
    ```bash
    cd [project-directory]
    ```
3. Install dependencies:  
    ```bash
    flutter pub get
    ```
4. Run the application:  
    ```bash
    flutter run
    ```
